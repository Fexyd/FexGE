# Some forms to organize the proyect

--- 

## SECCIONES

**En el proyecto hay varias secciones que pueden pertenecer desde cosas de bajo nivel como el corazón del Engine hasta simple GUI o UI**

---

#### **Las principales son:**

### **- Core**: 

Son los archivos principales que conforman el runtime y editor. si el editor y/o el runtime necesitan de esto, es de esta sección.

* **- Window**: Es la sección window. se encarga de todo lo relacionado a la creación y manipulacion de la ventana
* **- Input**: Se encarga de obtener y exponer funciones para usar el input
* **- Render**: Se encarga de obtener y exponer funciones para poder renderizar objs

### **- Runtime**: 

Es todo lo que conforma el runtime.

### **- Editor**: 

Es todo lo que conforma el Editor.

dependiendo a que sección pertenece tendra un nombre u otro
